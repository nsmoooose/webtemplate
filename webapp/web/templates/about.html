About this application

Information

This tool is developed with open source tools and the latest web standards are used when needed. Programing language is a scriptable language named Python. CherryPy is used to serve pages. This is a tiny and efficient web server.

Genshi is a powerfull templating engine. It renders html content based on template files. To validate content sent from the web client FormEncode is used. It ensures that the application is protected from injection attacks.

To store data SQLite is used. SQLite is a tiny database engine used by embedded applications. To simplify database access an OR mapper has been used. It is named SQLAlchemy. It maps between python objects and database tables. It also supports mapping between other database vendors than sqlite.

To make proper documentation Sphinx is used. It creates html documentation for the application and the source code.

Ensuring quality by using a test driven development approach. Automated tests must be run and for that job nose has been choosen. It finds all your unit tests and executes them with a coverage report. This so you can see what parts of your application that are tested and the bits that isn't tested. Twill is an application that lets you automate web requests and verification of the result that is part of the automated test suite. pylint is an excellent tool to use when finding common mistakes and make the source code consistent.
