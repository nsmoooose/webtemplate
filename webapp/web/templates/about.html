About this application

Information

This tool is developed with open source tools. Programing language is a scriptable language named Python. CherryPy is used to serve pages. This is a tiny and efficient web server.

Genshi is a powerfull templating engine. It renders html content based on template files. To validate content sent from the web client FormEncode is used. It ensures that the application is protected from injection attacks.

To store data SQLite is used. SQLite is a tiny database engine used by embedded applications. To simplify database access an OR mapper has been used. It is named SQLAlchemy. It maps between python objects and database tables.
